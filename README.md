# 🤖 SGD RAG · Sistema de Gestión Documental con IA

🚀 Plataforma inteligente que permite gestionar documentos y realizar consultas usando **Inteligencia Artificial (RAG)**.

---

## 🧠 ¿Qué hace este sistema?

SGD RAG combina gestión documental con IA para:

- 📂 Subir y organizar documentos
- 🔎 Buscar información dentro de archivos
- 🤖 Responder preguntas con IA
- 📄 Generar resúmenes automáticos
- 👥 Gestionar usuarios (admin / estándar)

---

## 🧩 Módulos principales

- 💬 **Asistente IA**
  - Responde preguntas sobre documentos
  - Usa contexto real (RAG)

- 📁 **Gestión de documentos**
  - Subida de archivos (PDF, DOCX, TXT, CSV, XLSX)
  - Indexación automática

- 👥 **Gestión de usuarios**
  - Roles: Administrador y Estándar

- 🔎 **Búsqueda inteligente**
  - Filtrado por documentos
  - Consulta precisa

---

## ⚙️ Tecnologías utilizadas

- 🐍 **Python + FastAPI**
- 🧠 **ChromaDB (vector store)**
- 🤖 **Sentence Transformers**
- 🔗 **LangChain**
- 🌐 **HTML + CSS + JavaScript**
